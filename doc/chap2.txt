  [1X2.1 [33X[0;0YClasses of abstract unitals[133X[101X
  
  [33X[0;0YBasic references on Hermitian polarities, Hermitian curves and the classical
  Hermitian  unital  are  [Hir98,  Section  II.5]  and  [BE08,  Chapter 2]. On
  Buekenhout's  construction  of  unitals  embedded in translation planes, see
  [BE08,  Chapter  4].  The  cyclic  unital  of order six is due to Bagchi and
  Bagchi [BB89].[133X

  [33X[1;0Y[29X[2XHermitianAbstractUnital[102X( [3Xq[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10YThe classical unital object, which is the abstract unital of order

  [33X[0;0YThe  Hermitian  curve  has  the  following canonical equation: [23XX_0^{q + 1} +

  unital with the help of [10XPGU(3,[3Xq[103X[10X)[110X and calls [10XAbstractUnitalByDesignBlocks[110X. The
  [10XName[110X of the unital is set as [10XHermitianAbstractUnital([3Xq[103X[10X)[110X.[133X
  
  [1X2.1-2 AllBuekenhoutMetzAbstractUnitalParameters[101X
  
  [33X[1;0Y[29X[2XAllBuekenhoutMetzAbstractUnitalParameters[102X( [3Xq[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10YAll   the   pairs   of   parameters   over  [23XGF(q^2)[123X  which  result
            non-isomorphic (orthogonal) Buekenhout-Metz unitals of order [3Xq[103X.[133X
  
  [33X[0;0YThe argument [3Xq[103X must be a prime power (if even, then at least 4).[133X
  
  [33X[0;0YIf  [3Xq[103X  is an odd prime power and [23X(\alpha, \beta)[123X is 2-tuple of [23XGF(q^2)[123X, then
  this  pair  is a suitable parameter of an orthogonal Buekenhout-Metz unital,
  if [23X(\beta^q - \beta)^2 + 4 \alpha^{q + 1}[123X is a nonsquare in [23XGF(q)[123X.[133X
  
  [33X[0;0YIf  [3Xq[103X is an even prime power and [23X(\alpha, \beta)[123X is 2-tuple of [23XGF(q^2)[123X, then
  this  pair  is a suitable parameter of an orthogonal Buekenhout-Metz unital,
  if  [23X\beta  \not\in  GF(q)[123X and [23X\alpha^{q + 1}(\beta^q + \beta)^2[123X has absolute
  trace 0.[133X
  
  [33X[0;0YIn  both  cases  [23X\alpha  = 0[123X yields the Hermitian classical unital, hence we
  omit the tuples with [23X\alpha = 0[123X.[133X
  
  [1X2.1-3 OrthogonalBuekenhoutMetzAbstractUnital[101X
  
  [33X[1;0Y[29X[2XOrthogonalBuekenhoutMetzAbstractUnital[102X( [3Xq[103X, [3Xalpha[103X, [3Xbeta[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10YThe  unital  object,  which  is  the  abstract  unital  of order [3Xq[103X
            isomorphic   to   the   orthogonal   Buekenhout-Metz  unital  with
            parameters [3Xalpha[103X and [3Xbeta[103X in the classical projective plane.[133X
  
  [33X[0;0YThe  argument  [3Xq[103X must be a prime power (if even, then at least 4), the other
  arguments  [3Xalpha[103X  and  [3Xbeta[103X  -  elements  of  [23XGF(q^2)[123X  - must be a pair from
  [10XAllBuekenhoutMetzAbstractUnitalParameters([110X[3Xq[103X[10X)[110X.[133X
  
  [33X[0;0YThe  point set [23XU_{\alpha, \beta} = \left\{ ( x, \alpha x^2 + \beta x^{q + 1}
  +  r,  1)  \colon x \in GF(q^2), r \in GF(q) \right\} \cup \left\{ (0, 1, 0)
  \right\}[123X  in  [23XPG(2,q^2)[123X  is  a unital (called the orthogonal Buekenhout-Metz
  unital)  if  the pair of parameters [23X(\alpha, \beta)[123X satisfies the conditions
  explained             in            the            description            of
  [10XAllBuekenhoutMetzAbstractUnitalParameters([110X[3Xq[103X[10X)[110X.[133X
  
  [1X2.1-4 BuekenhoutTitsAbstractUnital[101X
  
  [33X[1;0Y[29X[2XBuekenhoutTitsAbstractUnital[102X( [3Xq[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10YThe  unital  object,  which  is  the  abstract  unital  of order [3Xq[103X
            isomorphic   to   the  Buekenhout-Tits  unital  in  the  classical
            projective plane.[133X
  
  [33X[0;0YThe  argument  [3Xq[103X  must  be  a  power  of 2, such that the exponent is an odd
  integer  at  least  3. The point set [23XU_T = \left\{ ( x_0 + x_1 \delta, y_0 +
  (x_0^{\tau  +  2}  +  x_1^\tau  + x_0x_1)\delta, 1) \colon x_0, x_1, y_0 \in
  GF(q)\right\} \cup \left\{ (0,1,0) \right\}[123X in [23XPG(2,q^2)[123X is a unital (called
  the  Buekenhout-Tits  unital)  if  [23X\delta  \in  GF(q^2)  \setminus GF(4)[123X and
  [23X\delta^q  =  1 + \delta[123X. This [23X\delta[123X is just a basis element along with 1 in
  [23XGF(q^2)[123X  over  [23XGF(q)[123X, hence we can omit it as a parameter. The function [23X\tau
  \colon GF(q) \rightarrow GF(q)[123X assigns to the field element [23Xx[123X the following:
  [23Xx \mapsto x^{2^\frac{k + 1}{2}}[123X, where [23Xq = 2^k[123X.[133X
  
  [1X2.1-5 BagchiBagchiCyclicUnital[101X
  
  [33X[1;0Y[29X[2XBagchiBagchiCyclicUnital[102X( [3Xn[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10YA  unital  object  of  order  [10Xn[110X,  with a cyclic automorphism group
